Help

Use input field to add new tasks to the list
Hover over elements to toggle their methods
'Remove' button completely removes item from the list
'Insert' button brings the input field to the item, which allows you to add sub-tasks to it. Click on 'Your tasks' to bring the input form to the root of the list
'Expand/Collapse' button toggles sub-tasks that haven't yet been completed. It also shows its quantity
'Check/Uncheck' button toggles the state of the task. Completed and pending items are stored in their respective lists
'Show done/Hide done' button toggles sub-tasks that have been completed. It also shows its quantity. It is only shown if that item has completed tasks
